# 参考打法：女巫解药决策（今晚有刀口可救）

解药也是一次性资源，且整局只能救**当晚**刀口（救不了昨夜的、不能预留）。救与不救不是
"看到刀口就救"，而是一次"这个人值不值得我现在花掉唯一解药"的期望值判断。

## 值得救的局面

- **首夜（R1）默认倾向救**：首夜信息少、刀口往往看不出价值，但"留药等高价值目标"是个高风险赌注——
  **女巫是狼最优先想刀的目标之一，你 R1 留药，很可能 R2/R3 就被刀；过了首夜又不能自救，留的解药
  100% 作废。** 所以除非明显是"卖刀"（见下），首夜刀口就**倾向于救掉**，把解药真正用出去，
  而不是赌自己能活到更该救的那一夜。
- 刀口是**已公开且可信的预言家**（有清晰查验链、被场面认可）→ 救，保住好人核心信息源远比留药值。
- 刀口是发言强、能带队、好人方明显倚重的核心好人 → 救的边际收益高。
- 残局阶段，救一个好人能直接顶住胜负线（救完狼数仍 < 好人数）→ 救。
- **首夜刀口是你自己**：规则允许首夜自救；若你判断自己后续作用大、且暂无更该救的人，可自救
  （注意：第二夜起不能自救）。

## 该留药 / skip 的局面

- 刀口是身份不明、作用一般的普通玩家 → **仅在非首夜、且你确有理由相信自己能安全活到更该救的夜晚时**，
  才考虑留药。留药的代价是"被刀+不能自救=全废"，别为了一个"也许"的高价值救人机会，赌掉手里这瓶
  确定能用的药；尤其首夜不要轻易留（见上）。
- 你怀疑这是狼故意"卖刀"骗解药（比如刀一个发言差、本来就要被推的人）→ 谨慎，别为低价值目标
  花掉解药。
- 解药救人当晚不会暴露你，但**白天千万别因为"我知道他没死"而露出马脚**（见下）。

## 与毒药、规则的关系

- 默认规则同一晚**不能既救又毒**，最多用一瓶。若同时想救想毒，先比较两边收益，只能选其一。
- 救的是当前轮 `private_events.witch_kill_target_info` 给的刀口；历史刀口不能再救。

## 白天表达

- 救人后，白天发言仍要像普通平民，**绝不暗示"我知道谁昨晚被刀/被救了"**，否则等于告诉狼你是
  女巫、解药已用。守住身份比抢一句话的存在感重要。
