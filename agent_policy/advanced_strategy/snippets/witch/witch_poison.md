# 参考打法：女巫毒药决策

毒药是一次性资源，毒错好人比不毒更糟。把它当"高确定性的一次清狼机会"，按期望值判断，
不要凭模糊怀疑乱毒。

## 值得用毒的局面

- 某人被**公开查杀为狼**且仍存活、白天又难以靠投票放逐他（票型被狼带偏）→ 毒药补刀价值高。
- 残局票数贴胜负线，毒掉一个高置信狼能直接守住或扭转 → 即使略激进也值。
- 你有强逻辑链锁定某人是狼（发言自爆、查杀对跳里站错边等公开证据），而非仅凭直觉。
- 目标同时满足"公开证据强"和"仍然存活且白天未必能稳定推出"时，毒药收益最高。

## 该留药/skip 的局面

- 早期信息不足、全场只有模糊怀疑 → **skip**，把毒药留到证据清晰时。
- 没有任何公开狼读时不要赌，毒错一个好人等于帮狼减员。
- 仅有主观嫌疑排序、跟风票或一句发言状态差，不足以单独支撑用毒。
- 当晚有值得救的刀口、且规则不允许同夜救毒时，先权衡救人与毒人的收益，只能选其一。

## 目标约束

- 只毒存活、非自己、有较高狼嫌疑的目标；绝不毒死亡玩家或明显可信好人。

## 白天表达

- 用过毒之后，白天发言仍要表现为谨慎好人，不要暴露你是女巫、也不要泄露夜晚刀口信息。
