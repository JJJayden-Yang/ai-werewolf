# 参考打法：多预言家对跳（场上 ≥2 人跳了预言家）

已经有两个或更多人公开跳预言家——其中**至少一个是狼悍跳**。这一刻全场最值钱的事，是把
"谁是真预言家"这条线判断清楚并据此归票；最坏的打法是两边都不敢信、把票投到对跳之外的人，
等于放着已经暴露的狼不抓。

## 如果你不是预言家（平民 / 女巫 / 猎人）

这是你最该认真读盘的时刻，别置身事外说"等他们对完"。可比较的公开维度：

- **查杀/金水指向谁**：两边各自报了谁是狼、谁是金水？被报狼的人后续反应是否自然？
- **查验逻辑是否自洽**：首验选人理由、报验时机（D1 就跳还是被逼跳）、前后发言有没有预言家该有的视角。
- **谁在帮哪个预言家**：注意有没有人异常积极地保某一个跳预言家的人、或急着踩另一个——
  狼常会用同伴的票和发言给悍跳"染色"。
- **被发金水的人立场**：被某预言家发了金水的人，是顺势帮他、还是反过来怀疑他？金水反水是重要信号。

读完给出**明确站边**（哪怕是低置信的倾向）并说明依据，把票往"你判断的那个假预言家 / 他报的狼"
集中。**别和稀泥**——对跳里必有一狼，全场都弃权才是狼最想要的结果。

## 如果你被卷进对跳（你自己是真预言家或被冒名）

- 真预言家：以你的角色基础策略为准，坚持报清查验链、对查到底，不要被带乱节奏。
- 投票时围绕"两个预言家里哪个更可能是狼"组织，不要把焦点散到第三方。

## 投票（若进入投票阶段）

- 优先把票集中到你判断为**假预言家**本人，或他报出的"查杀"（若你认为发查杀的才是真预言家）。
- 避免再平票送掉信息：对跳局里票型分散最有利于狼，明确归票比"再观察"更重要。
