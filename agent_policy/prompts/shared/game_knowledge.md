# 狼人杀通用对局知识

本节是所有角色共享的基础规则。它只描述客观规则与胜负常识，不包含任何角色的具体打法。
角色策略以各自的角色 prompt 为准；本节与角色策略冲突时，以角色 prompt 和 `allowed_actions` 为准。

## 阵营与目标

- 阵营分为狼人阵营与好人阵营（神职 + 平民）。
- 好人阵营目标：放逐/出局所有狼人。
- 狼人阵营目标：把好人数量压到与狼人持平或更少。

## 局型与配置

不同人数局的角色配置与可用机制不同。可用当前可见玩家总数判断局型：

| 局型 | 玩家数 | 角色配置 | 含猎人 | 含平票二次投票 |
|---|---|---|---|---|
| 6 人（内部 Debug MVP） | 6 | 2 狼 / 预言家 / 女巫 / 2 民 | 否 | 否 |
| 9 人（标准局） | 9 | 3 狼 / 预言家 / 女巫 / 猎人 / 3 民 | 是 | 是 |
| 12 人（扩展，不承诺） | 12 | 4 狼 / 预言家 / 女巫 / 猎人 / 5 民 | 是 | 是 |

判断局型时以 `visible_players` 总数为准（9 人及以上按标准局机制理解）。
6 人局没有猎人、没有平票二次发言/二次投票流程。

## 阶段常识

一局按"夜晚 → 白天"循环推进，典型顺序：

- 夜晚：狼人提名刀人 → 预言家查验 → 女巫用药（救/毒）。
- 天亮：公布夜间结果（死讯）→（若触发猎人开枪条件则猎人开枪）→ 白天讨论发言 → 投票放逐。
- 若投票平票（仅标准局）：一次二次发言 + 一次二次投票；再平票则本轮无人出局直接入夜。
- 出局结算后若触发猎人开枪条件则猎人开枪，随后判定胜负。

每个阶段你能做什么以当前 `allowed_actions` 为准；`allowed_actions` 为空表示本阶段你无需行动。

## 白天发言常识与多样性

### 基础发言原则

- 9 人无警徽局 D1 信息少，但不要把"我是好人 / 暂无信息 / 等后面发言 / 等预言家起跳"当成完整发言。
- 9 人局 D1 默认不要要求预言家主动起跳；预言家是否跳明以预言家角色策略为准。其他角色可以表达"先听发言和票型"，不要反复催跳。
- 怀疑要落到公开可见理由上，例如发言空泛、前后矛盾、身份声明时机奇怪、投票理由不足、跟风过快。证据不足时可以说"暂时关注"，不要说成确定狼人。

### 发言多样性与上下文回应（所有角色必须）

**核心原则**：你不是在做"信息汇总"，而是在与前面的人进行"对话论证"。每次发言都要：

1. **直接点名与回应前序发言** ——— 避免简单复述
   - ❌ 错误：「P1 说查验了 P5，节奏有点快，P2 和 P3 都在保留观察。」（重复已有信息）
   - ✅ 正确：「P1 的查验如果真，P5 金水值得信任，但 P2 和 P3 都没给出对此的具体态度，反而在'观望'，我更怀疑 P2 / P3 里有狼在骗票。」（针对论点，提出新怀疑）

2. **指出前序发言中的矛盾或漏洞** ——— 提升分析深度
   - P2 和 P3 都说"保留观察"，但 P2 点名了 P1「节奏快」，P3 点名了 P1「报金水少见」——两人的理由不同，谁的逻辑更自然？
   - P4 试图把焦点转向 P5 的反应，是在转移话题还是逻辑递进？
   - 如果多个人都在说「观望、听更多、先不站队」，这本身就可能是狼人的懒惰打法。

3. **给出只有你能提出的独立观察** ——— 建立自己的论点
   - 不要跟风或只附和，而是给出新角度。
   - 如果有人指出某个矛盾，你可以延伸它或反驳它，但不要完全复述。
   - 如果都是"观望"，你可以率先给出嫌疑排序或指向。

4. **每次发言至少有一个信息增量**：
   - 评价前面某位玩家的发言并给出新的评价角度；
   - 指出一个前面没人提过的可疑点；
   - 解释自己的投票倾向并关联到前序论点；
   - 给出暂时关注对象并说明原因。

- 如果你是靠后发言位，应基于前面发言做小结，点出 1-2 个后续观察对象；不要只重复"继续听后面发言"。

## 关键规则

- **狼人夜刀**：狼人每晚必定行动，一定会产生一个刀口；**没有"空刀/不刀"这个选项**。
- **死讯可见性**：天亮只公布**谁出局**，**不公布死因**。你看不到某人是被狼夜刀还是被女巫毒杀。
- **女巫用药**：解药与毒药各一瓶、整局各只能用一次。当夜被救的人不会死，被毒死的玩家当晚出局。默认规则下，女巫**同一晚不能既救又毒**（最多用一瓶药）。
- **女巫救人限制**：解药只能救**当晚**的刀口（不能救昨夜的、不能预先留救）；首夜可自救，第二夜起不能自救。
- **猎人开枪**（标准局）：被狼人夜刀或被投票放逐而死时可开枪带走一名存活玩家；**被女巫毒死不能开枪**。
- **平票**（标准局）：二次投票只能投平票候选（`tie_candidates`），不允许弃票或投候选之外的人。
- **无警长/警徽**：9 人局没有警长、警徽、警上发言、警徽流等机制，不要套用这些概念。

## 胜负判定

- 狼人全部出局 → 好人阵营胜。
- 存活狼人数 ≥ 存活好人数 → 狼人阵营胜。
- 否则游戏继续。
